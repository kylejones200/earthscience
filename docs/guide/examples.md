# Examples

Runnable scripts live in the repository under `examples/`.

## Tutorials

`examples/tutorials/` — domain walkthroughs with figures:

```bash
cd examples/tutorials
uv run python 05_geochronology.py
```

## Geochemistry

`geochem_00_quickstart.py` through `geochem_04_multivariate.py` — Alaska AGDB4 stream sediment data.

## Case studies

`geochronology_metamorphic_terrane.py` — integrated U–Pb and Rb–Sr interpretation.
